Easy Installation for PyMVPA

We recently held a workshop for PyMVPA and found that installation can be challenging on non-Linux systems. To simplify the process, we recommend using pip, Python's package installer, to install PyMVPA directly from the Python Package Index (PyPI).

Installation Steps

  1. Ensure that you have Python installed on your system. PyMVPA is compatible with Python 2.7 and Python 3. If you don't have Python installed, download and install the latest version suitable for your operating system.
  2. Open your command line interface:
    • Windows: Command Prompt or PowerShell
    • macOS: Terminal
    • Linux: Terminal
  3. Upgrade pip to the latest version to ensure compatibility: 
    pip install --upgrade pip
  4. Install PyMVPA using pip: 
    pip install PyMVPA
  5. Verify the installation by importing PyMVPA in a Python session: 
    python -c "import mvpa2; print(mvpa2.__version__)"
    This command should display the installed version of PyMVPA.

Additional Dependencies

PyMVPA relies on several external packages for full functionality. If you encounter any issues or require additional features, consider installing the following packages:

pip install numpy scipy matplotlib scikit-learn nibabel

These packages provide support for numerical computations, scientific computing, plotting, machine learning, and neuroimaging data formats, respectively.

Troubleshooting

  • If you encounter permission errors during installation, try adding --user to the pip commands: 
    pip install --user pymvpa2
  • For systems with multiple versions of Python, you may need to use pip3 and python3 instead of pip and python to specify Python 3: 
    pip3 install PyMVPA
python3 -c "import PyMVPA; print(PyMVPA.__version__)"
  • If you experience issues related to package versions or dependencies, consider creating a virtual environment to manage your Python packages in isolation. Refer to the official Python documentation for guidance on setting up virtual environments.

For more detailed information and support, visit the official PyMVPA installation guide.
